# Finance Dashboard

## Overview
A simple finance dashboard built using HTML, CSS, and JavaScript to track financial activity.

## Features
- Dashboard summary (Balance, Income, Expenses)
- Spending breakdown chart
- Transactions list with search
- Role-based UI (Viewer/Admin)
- Insights (monthly comparison, top category)

## Role Functionality
- Viewer: Can only view data
- Admin: Can add transactions

## How to Run
Open index.html in any browser

## Tech Used
- HTML
- CSS
- JavaScript

## Approach
Focused on simplicity, readability, and clear UI structure while implementing all required features.
